# Autonomous Controllable Car

## Overview

The goal of this project is to create a fully autonomous car that can also be controlled manually if desired. REMEMBER TO INCLUDE VIDEO.

## Parts List
* [mbed LPC1768][mbed]
* [Long Breadboard][LBB]*
* [Short Breadboard][SBB]*
* [Shadow Chassis][SC]
* [Shadow Chassis Motor][SCM]
* [Wheels][WH]
* 2x [Battery Holder][BH]
* [Motor Driver][MD]
* [Bluetooth Sensor][BS]
* [Speaker][SP]
* [Audio Amp][AA]
* [uLCD][ULCD]
* [Lidar Sensor][LS]
* 8x AA Batteries
* Wires

*Different breadboard sizes can be used, these two provided our team adequate space.
